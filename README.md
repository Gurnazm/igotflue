igotflue – Juno Records Plak Fiyat Listeleyici

Bu proje, juno.co.uk adlı yurt dışı plak sitesinde yer alan belirli plakların fiyatlarını otomatik olarak çekerek igotflue.com için listeleyen Python tabanlı bir programdır.

Amaç, aranan plakların güncel fiyat bilgilerini merkezi ve düzenli bir şekilde sunmaktır.

Özellikler

Juno Records (juno.co.uk) üzerinden plak fiyatlarını çekme

Aranan plaklara göre veri toplama

Fiyat bilgilerini liste formatında sunma

Genişletilebilir ve güncellenebilir yapı

Kullanılan Teknolojiler

Python

Web scraping

HTTP istekleri

HTML / JSON veri ayrıştırma
